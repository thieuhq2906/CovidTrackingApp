Download the application includes library folder, runnable execute file, 
Before running the application, the users need to download the additional libraries as: 
pip install cz_Freeze
pip install PyQtWebEngine
pip install matplotlib
pip install pandas
pip install geopandas
pip install seaborn
pip install mplcursors
pip install folium 
Then, the users just run the file CovidTrackingApp.py to start the app.
Or they can run: python setup.py build to make the execution file. Then, run that file
